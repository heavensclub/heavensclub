## 🚀 Deploy to GitHub Pages

1. Create a new repository on GitHub named `Heavensclub`
2. Push this code:
   ```bash
   cd Heavensclub
   git init
   git add .
   git commit -m "Initial commit - Heavens Club Website"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/Heavensclub.git
   git push -u origin main
   ```
3. Go to **Settings → Pages → Source** → Select `main` branch → Save
4. Your site will be live at `https://YOUR_USERNAME.github.io/Heavensclub`
